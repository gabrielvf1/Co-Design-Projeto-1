Descrição do projeto:

  • Nesse projeto deverímos modelar um sistema Termodinâmico. O sistema escolhido pela dupla foi de um carro queimando pneu.
  • Esse foi o primeiro projeto em que utilizamos Equações Diferenciais para que nossos resultados se aproximassem da realidade
  • Com essa modelagem conseguimos respoder se é possível que um carro exploda os seus pneus ao rodá-los em falso.
  • Através dos códigos conseguimos fazer uma comparação entre carros com diferentes potências e pesos, para ver qual a influência dessas variáveis no modelo
