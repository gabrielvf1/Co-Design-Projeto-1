Descrição do projeto:

  • O projeto envolveu criar um circuito astável (pisca-pisca).
  • Ele foi o primeiro projeto feito no FabLab do Insper. Nele aprendemos noções básicas de eletrônica.
  • Além disso, aprendemos como as placas eletrônicas eram usinadas no FabLab.
  • O produto final foi um pequeno circuito com dois LED's que acendiam intercaladamente.
  • Mesmo sendo simples esse projeto foi muito importante pois nos deu noção de como o FabLab funcionava e para quê ele funcionava. Esse projeto abriu portas para explorarmos cada vez mais o que o FabLab tinha a oferecer.
